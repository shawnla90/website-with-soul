### How blog posts work

1. Create a new `.md` file in `content/blog/`
2. Add frontmatter (title, date, excerpt, category)
3. Write your content in markdown
4. The site picks it up automatically on next build

### Frontmatter options

```yaml
---
title: "Your Post Title"
date: "2026-01-15"
excerpt: "A short description for cards and SEO."
category: "builds"
featured: true
---
```

### What to write about

Write about what you're building. What you're learning. What broke and how you fixed it. The posts that perform best are the ones that sound like you, not like a content mill.
