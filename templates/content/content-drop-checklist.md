# Content Drop Checklist

> Use this checklist every time you distribute a blog post across platforms. Work through it sequentially. Do not post until all review steps are complete.

## Source

- [ ] Source blog post is published and live on your website
- [ ] URL is working and accessible
- [ ] OG image generates correctly (check with a link preview tool)
- [ ] RSS feed includes the new post

## Draft Generation

- [ ] Reddit draft created (adapted per subreddit, conversational tone)
  - Subreddit 1: [FILL IN: e.g. r/webdev] - Draft at: `content/reddit/drafts/`
  - Subreddit 2: [FILL IN] - Draft at: `content/reddit/drafts/`
  - [Add more subreddits as needed]
- [ ] LinkedIn draft created (builder voice, whitespace-heavy, structured)
  - Draft at: `content/linkedin/drafts/`
- [ ] X thread draft created (punchy standalone tweets, 5-7 tweets)
  - Draft at: `content/x/drafts/`

## Review (Do Not Skip)

- [ ] Reddit draft reviewed against subreddit rules
- [ ] Reddit draft reviewed for tone (sounds like a community member, not a marketer)
- [ ] Reddit draft has no hard CTA or direct link in the body
- [ ] LinkedIn draft reviewed for voice (sounds like you, not a chatbot)
- [ ] LinkedIn draft has proper paragraph breaks and whitespace
- [ ] X thread reviewed. Each tweet stands alone. Hook in tweet 1.
- [ ] All drafts ran through anti-slop checklist
- [ ] All drafts checked for accuracy (links, code, claims)

## Finalize

- [ ] Reddit final version saved to `content/reddit/final/`
- [ ] LinkedIn final version saved to `content/linkedin/final/`
- [ ] X final version saved to `content/x/final/`

## Post

- [ ] Reddit posted to subreddit 1
- [ ] Reddit posted to subreddit 2
- [ ] LinkedIn posted
- [ ] X thread posted

### Spacing

Do not post everywhere at the same time. Stagger across 1-2 days:

- Day 1: Reddit (primary subreddit)
- Day 1 (afternoon): LinkedIn
- Day 2: X thread
- Day 2 (afternoon): Reddit (secondary subreddit, if relevant)

This avoids looking like a content blast and gives you time to engage with early responses on each platform before the next one goes live.

## Post-Distribution

- [ ] Monitor Reddit comments for 24 hours. Reply to every comment.
- [ ] Monitor LinkedIn comments. Reply within a few hours.
- [ ] Monitor X replies. Engage with meaningful responses.
- [ ] Note what resonated and what fell flat in `tasks/lessons.md`
- [ ] Update content strategy based on performance
